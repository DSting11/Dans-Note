⚙ Perlu Setup Firebase (sekali saja) Ganti nilai firebaseConfig di bagian script dengan config dari project Firebase kamu.
⚠ Tidak bisa terhubung ke Firebase Kemungkinan besar Database Rules perlu diubah. Ikuti langkah ini:
1. Buka console.firebase.google.com
2. Pilih project → Realtime Database → tab Rules
3. Hapus semua isinya, ganti dengan ini lalu klik Publish:
{ "rules": { ".read": true, ".write": true } }
4. Refresh halaman ini ✓
